# 介绍
chatafl的种子丰富模块的复现，以及适度微调。

# 使用
```
python run.py
```
自动加载config.yml配置文件，根据大模型温度、最大添加消息类型数量、最大消息语料数量、协议、输入路径、输出路径、单个种子生成数量等需求批量生成种子。
